# 🤖 AI QA Assistant

AI QA Assistant is an intelligent tool that automatically generates **test cases and edge cases** from software requirements using AI.

Designed and implemented backend APIs for processing input and generating structured outputs using LLMs.

---

## 🚀 Features

* Generate functional test cases from input requirements
* Identify edge cases (boundary, invalid, security)
* Automate QA workflows using LLMs
* Export structured test cases

---

## 🛠 Tech Stack

* Python (Flask)
* OpenAI API
* Pandas (for formatting output)

---

## 💡 Use Case

Helps developers and QA engineers quickly generate test cases, reducing manual effort and improving testing coverage.

---

## 🚀 Highlight

AI-powered automation tool designed for real-world QA workflows and testing systems.
